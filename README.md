# 프로젝트 이름
    키움 RestAPI 사용하고 만원 받자

## 사전설치
   python 설치(공통)
   
       brew install python (mac)
       brew install python-tk (mac)
       
   키움RestAPI 사용 신청
   
       https://openapi.kiwoom.com/intro/serviceInfo
## 설치하기
    실행에 필요한 모듈 설치
    make install
## 실행    
    make run
## 바이너리 생성하기
    make dist
    참고로 dist폴더에 window, mac 실행 파일 존재함.
